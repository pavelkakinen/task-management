# Task Management System

A web-based task management application built with PHP and MySQL, featuring full CRUD functionality and a responsive user interface.

##  Features

-  Create, read, update, and delete tasks
-  User-friendly interface with responsive design
-  MySQL database integration
-  Secure data handling with PDO
-  Task filtering and search functionality
-  Priority levels and status tracking

##  Technologies

- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Architecture:** MVC-inspired structure
- **Security:** PDO prepared statements for SQL injection prevention

##  What I Learned

- Building database-driven web applications
- Implementing secure database interactions with PDO
- Structuring PHP projects with separation of concerns
- Working with CRUD operations
